# SMS Spam Classification

## 📌 Objective
This project classifies SMS messages as Spam or Ham using NLP techniques.

## ⚙️ Methods Used
- Bag of Words (BoW)
- TF-IDF

## 🤖 Model
- Logistic Regression

## 📊 Results
- BoW Accuracy: ~95%
- TF-IDF Accuracy: ~97%

## 📁 Dataset
SMS Spam Collection Dataset

## 🚀 Conclusion
TF-IDF performs better than Bag of Words for spam detection.

## 🔗 GitHub
This project demonstrates text preprocessing, vectorization, and classification using machine learning.
